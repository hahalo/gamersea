## 在线云分享

游民海洋是一个基于laravel 5.1 、redis、mysql、SQLSERVER开发的一个分享网站、其中包含云相册、文章、日志、vip视频、在线聊天、社工库、游戏商城功能分享

## 首页

分享的图片直接显示在首页

## 海洋商场

游戏商城分享，需要安装SQLSERVER数据库（配置文件默认屏蔽）

## 海洋角色

游戏角色分享，需要安装SQLSERVER数据库（配置文件默认屏蔽）

## 云文章

分享的文章都在这里显示

## 云日志

分享每天的工作心得

## 云相册

可以上传自己的相册
